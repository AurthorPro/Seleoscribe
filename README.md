# Luminary — Cosmic Journal

A local-first cosmic journaling app. Journal guided by moon phases, planetary transits, and your personal birth chart.

## Deploy

This is a single-file PWA (`index.html`). To deploy:

**GitHub Pages:**
1. Push `index.html` to a GitHub repo
2. Go to Settings → Pages → Source → main branch / root
3. Your app will be live at `https://yourusername.github.io/reponame`

**Custom domain:**
- Add a `CNAME` file with your domain name to the repo root

## Features
- Daily affirmation with goal-directed AI generation
- Moon phase tracking with 8-phase lunar cycle
- Planetary transit guidance (2026 curated dataset)
- AI-powered journaling prompts and cosmic guide chat
- Full calendar with cosmic events and custom event adding
- Birth chart profile with personalized AI context
- Journal frequency goals and streak tracking
- Memory system — learns your writing voice over time
- Multi-provider AI support: Groq (free), Claude, GPT, Gemini, Grok
- Export journal as JSON or Markdown
- PWA — installable on iPhone/Android as a home screen app
- All data stored locally on device, never transmitted
