# Luminal — Cosmic Journal

A local-first cosmic journaling app. Journal guided by moon phases, planetary transits, and your personal birth chart.

## Deploy to GitHub Pages

1. Push contents of this folder to a GitHub repo root
2. Go to Settings → Pages → Source → main branch → / (root)
3. Live at `https://yourusername.github.io/reponame`

## Features
- Daily affirmation with AI generation
- Moon phase tracking with 8-phase lunar cycle
- Planetary transit guidance
- AI-powered journaling prompts and cosmic guide chat
- Calendar with cosmic events and custom event adding
- Birth chart profile with personalized AI context
- Journal frequency goals and streak tracking
- Memory system — learns your writing voice over time
- Multi-provider AI: Groq (free), Claude, GPT, Gemini, Grok
- Export journal as JSON or Markdown
- PWA — installable on iPhone/Android
- All data stored locally, never transmitted
